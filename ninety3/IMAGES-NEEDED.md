# Images Required for Ninety 3 Website

## ⚠️ IMPORTANT: Images Need to Be Added

The website structure is complete and pushed to GitHub, but you need to add the actual image files.

## Required Images

Add these images to the `ninety3/images/` folder:

1. **logo.png** - Your Ninety3 cursive script logo (white on transparent/black)
2. **person-green-tracksuit.jpg** - Person wearing dark green Ninety3 tracksuit
3. **black-sentizero-tracksuit.jpg** - Black SENTIZERO tracksuit full set
4. **black-sentizero-hoodie.jpg** - Black SENTIZERO hoodie only
5. **black-sentizero-pants.jpg** - Black SENTIZERO sweatpants only

## How to Add Images

1. Copy your image files to the `ninety3/images/` folder
2. Make sure the file names match exactly (case-sensitive)
3. Commit and push:
   ```bash
   git add ninety3/images/*.png ninety3/images/*.jpg
   git commit -m "Add Ninety 3 brand images"
   git push
   ```

Once the images are added, they will automatically appear on the website!
