# Ninety 3 - Clothing Brand Website

A modern, dark-themed website for Ninety 3 clothing brand featuring the SENTIZERO Edition collection.

## Features

- **Dark Theme**: Sleek, modern dark interface
- **Responsive Design**: Works perfectly on all devices
- **Gallery Section**: Showcase of people wearing the brand
- **Product Page**: Dedicated SENTIZERO Edition products page
- **Smooth Animations**: Modern scroll effects and interactions
- **Mobile Menu**: Responsive navigation

## Image Setup

The website is already configured to use your images! Just add them to the `images` folder:

### Required Images:

1. **logo.png** (or logo.jpg)
   - Your Ninety3 cursive script logo
   - White logo on transparent or black background
   - Place in: `ninety3/images/logo.png`

2. **person-green-tracksuit.jpg**
   - Image of person wearing dark green Ninety3 tracksuit
   - Place in: `ninety3/images/person-green-tracksuit.jpg`

3. **black-sentizero-tracksuit.jpg**
   - Main product: Black SENTIZERO tracksuit (full set)
   - Place in: `ninety3/images/black-sentizero-tracksuit.jpg`

4. **black-sentizero-hoodie.jpg**
   - Black SENTIZERO hoodie only
   - Place in: `ninety3/images/black-sentizero-hoodie.jpg`

5. **black-sentizero-pants.jpg**
   - Black SENTIZERO sweatpants only
   - Place in: `ninety3/images/black-sentizero-pants.jpg`

All images are already referenced in the HTML files - just add your image files to the `images` folder and they'll appear automatically!

See `images/README.txt` for detailed instructions.

## Customization

- **Colors**: Edit the CSS variables in `styles.css` (lines starting with `--`)
- **Content**: Update text in `index.html` and `products.html`
- **Pricing**: Modify prices in `products.html`

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Contact Information

Update the contact section in `index.html` with your actual:
- Email address
- Social media handles
- Location/Shipping info
